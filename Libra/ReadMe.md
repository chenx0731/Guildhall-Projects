CONTROLS

- Tank Turn & Move: ESDF / left joystick
- Turret Turn & Move: IJKL / right joystick
- Fire: space bar / A or right trigger
- Pause: P / Start
- Exit: ESC / Back
- Slow mode: T
- Debug mode: F1
- No clip: F3
- Debug camera: F4
- Heat map: F5
- Restart game: F8
- Move to next map: F9
